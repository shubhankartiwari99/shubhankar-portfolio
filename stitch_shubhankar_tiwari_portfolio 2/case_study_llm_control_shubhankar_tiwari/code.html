LLM Control System

Runtime Reliability for Generative Models

Built a full-stack system to monitor and control LLM generation at the token level, with adaptive instability detection and real-time intervention.

code View Repo

Core Insight

Detect and correct unstable token generation in real time using entropy-based control.

The system observes the generation loop manually, extracting per-token probability and entropy. This allows for mid-generation intervention without full restarts.

  • check_circle Token-level observability via manual decoding loop.
  • check_circle Adaptive regeneration and temperature adjustment.

radar Stability Detector System

signal_01

entropy_collapse

Sudden drop in entropy indicating model certainty is artificially high, often leading to repetitive or degenerate output.

signal_02

repetition_loop

Detected cycling of n-grams over a sliding window, breaking the generation flow.

signal_03

low_entropy_lock

Sustained period of low entropy indicating model is stuck in a highly deterministic, likely uncreative path.

gavel Policy Controller & Performance

A formal policy controller implements explicit intervention rules mapped directly to the instability signals. When a signal trips, the controller adjusts parameters like temperature mid-generation or triggers localized token regeneration.
